Introduction

HTML is the standard markup language for creating Web pages. It stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. It describes the structure of a Web page. HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

Different Types of Tags

  • Heading Tags: <h1> to <h6>
    Heading tags that define headings, where <h1> is the largest and <h6> is the smallest.
    Example:-

    Code:-

    <h1>First Heading</h1>
    <h6>Last Heading</h6>

    Result:-

    First Heading

    Last Heading

  • Paragraph Tag: <p>

    • Represents a paragrph of text.
    • Its adds space before and after the text block.

    Example:-

    Code:-

    <p>This is the Report for the Internet Programing Cource. </p>

    Result:-


    This is the Report for the Internet Programming Cource.

  • Anchor Tag: <a> (used for links)

    • The <a> tag in HTML is used to create hyperlinks, which are clickable links that allow users to navigate to other web pages, sections of the same page, email addresses, or downloadable files.

    Example:-

    Code:-


    <a href ="https://www.w3schools.com/html/html_basic.asp">w3schools</a>

    Result:-

    w3schools

  • Image Tag: <img>

    • The <img> tag in HTML is used to embed images into a web page. It is a self-closing tag, meaning it does not have a closing tag like <img></img>.

    Example:-

    Code:-

    <img src="./Premier_logo.png" alt="University Logo" style="width: 100px; height: auto; margin: 10px 0;">

  • List Tags: <ul>, <ol>, <li>

    • <ul>:-The <ul> tag is used to create an unordered list where items are displayed with bullet points.

    Code:-

    <li>:-<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
    </ul>

    Result:-

    • Item 1
    • Item 2
    • Item 3
    • <ol>:-The <ol> tag is used to create an ordered list where items are displayed with numbers or other markers.

    Code:-

    <ol>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
    </ol>

    Result:-

    1. First Item
    2. Second Item
    3. Third Item
    • <li>:-The <li> tag is used to define each list item inside <ul> or <ol>.

    Code:-

    </ul>

  • Table Tags: <table>, <tr>, <td>, <th>

    • <table>:-The <table> tag is the container for creating a table in HTML.
    <table> <!-- Table rows and data go here --> </table>>

    • <tr>:- The <tr> tag is used to define a table row. A table can have multiple rows.
    <table border="1">
    <tr>
    <!-- Table cells go here -->
    </tr>
    </table>

    • <td>:-The <td> tag defines table data cells. These are the actual cells where the content is placed.

    Code:-

    <table border="1">
    <tr>
    <td>Row 1, Cell 1</td>
    <td>Row 1, Cell 2</td>
    </tr>
    <tr>
    <td>Row 2, Cell 1</td>
    <td>Row 2, Cell 1</td>
    </tr>
    </table>

    Row 1, Cell 1 Row 1, Cell 2
    Row 2, Cell 1 Row 2, Cell 2

    • <th>:-The <th> tag defines header cells in a table. Headers are bold and centered by default.

    Code:-

    <table border="1">
    <tr>
    <th>Header 1</th>
    <th>Header 2</th>
    </tr>
    <tr>
    <td>Data 1</td>
    <td>Data 2</td>
    </tr>
    </table>

    Header 1 Header 2
    Data 1 Data 2

  • Semantic Tags: <header>, <section>, <article>

    Semantic tags provide meaning to the structure of a webpage. Examples include:

    • <header>: Defines a header section.
    • <section>: Defines sections of content.
    • <article>: Represents independent content.

    Examples:

    Header Example:

    Welcome to My Website

    Your go-to place for learning HTML

    Section Example:

    About Us

    This section provides information about our website and its purpose.

    Article Example:

    Latest News

    HTML5 has introduced new semantic tags to make web pages more meaningful and accessible.


  • Inline Tags: <span>, <strong>

    Inline tags are used to style or format parts of text without breaking the flow of content. Examples include:

    • <span>: Used for styling specific parts of text inline.
    • <strong>: Used to emphasize text, typically making it bold.

    Examples:

    Span Example:

    This is a normal sentence, but here is a highlighted part using the <span> tag with custom styling.

    Strong Example:

    This is an important message: Please read carefully!
